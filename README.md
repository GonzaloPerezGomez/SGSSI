
# Docker LAMP
Linux + Apache + MariaDB (MySQL) + PHP 7.2 on Docker Compose. Mod_rewrite enabled by default.
docker-compose version 1.29.2

## Instrucciones
Introduce los siguientes comandos para empezar:
```bash
$ sudo rm -r mysql
$ docker build -t web .
$ docker-compose up
```

Para detenerlo:
```bash
$ docker-compose stop
```
